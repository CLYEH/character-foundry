## Notes

- React 19 + RR v7 的配對用 `react-router` 套件（v7 後套件重命名）
- Tailwind 4 的 config 很簡潔（CSS-first），不要照抄 v3 教學
- shadcn 的 `components.json` 用 `"style": "new-york"` 或 `"default"`（擇一，之後難改）
- Sonner 是 shadcn 新版 toast 首選（取代舊 `useToast`）
- 先不做 Inpaint canvas / react-konva（Sprint 3 才會用）
- 先不做 fetch-event-source（Sprint 2 真的要 SSE 時再加）
